                              Admin Mod platoon2 • 2 years ago

                              Please check http://www.makeinjava.com/c...

                              1. We have used Jersey Request Response filters.
                              2. When request hit the REST service, we note down the current time and put in MDC
                              MDC.put("start-time", String.valueOf(System.currentTimeMillis()));

                              Refer Class: CustomLoggingFilter from above mentioned link.
                              3. When request executed and we are going to generate the response then we will extract the startTime (which we had put to MDC) and calculate the total request execution time.

                              String stTime = MDC.get("start-time");
                              long executionTime = System.currentTimeMillis() - startTime;

                              Refer Class: CustomLoggingFilter from above mentioned link.

                              Similarly, we can use MDC to generate unique UUID, when request enters the service and generated UUID can be used in corresponding REST resources.
